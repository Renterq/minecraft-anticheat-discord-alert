# 🛡️ Minecraft Anti-Cheat Discord Alert Bot (English)

This project is a Python-based Discord bot that monitors a Minecraft server's log file (`latest.log`) in real-time and automatically forwards suspicious activity alerts from Anti-Cheat plugins like "Vulcan" or "GrimAC" directly to a designated Discord channel.

## 🚀 Features
- **Real-time Log Monitoring:** Checks the Minecraft server's log file every 5 seconds for new entries.
- **Smart Filtering:** Captures only the lines containing the keywords "Vulcan", "Grim", or "GrimAAC", which are generated by the anti-cheat plugins.
- **Discord Integration:** Forwards the detected suspicious activities as a formatted, attention-grabbing message (🚨 **Hile Uyarısı!** / **Cheat Alert!**) instantly to the Discord server.
- **Automatic Position Tracking:** Remembers its last read position in the log file to prevent redundant reading and save system resources.

## 🛠️ Installation
1. Ensure Python 3.8 or higher is installed.
2. Install the required library: `pip install discord.py`
3. Open `main.py` and configure the following variables according to your system:
   - `TOKEN`: Your bot token from the Discord Developer Portal.
   - `CHANNEL_ID`: The ID of the Discord channel where you want the logs to be sent.
   - `LOG_FILE_PATH`: The full path to your Minecraft server's log file (e.g., `/home/minecraft/server/logs/latest.log`).
4. Start the bot: `python3 main.py`
